Along with the ability to bundle all the routes together, we can also optionally bundle specific parts of the application together into chucks we can grab on-the-fly when the user goes to specific routes!

In fact... this very page was requested separately from the main app.js. Check the network tab to see this in action (look for the 1.chunk.js file)!

This is great for large apps where we don't need or want the user to download the entirety of the application code at once.
